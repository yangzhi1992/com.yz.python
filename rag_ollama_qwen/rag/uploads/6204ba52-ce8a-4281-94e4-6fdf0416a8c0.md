# Spring Boot 知识库

Spring Boot 是一个基于 Java 的微服务框架，由 Pivotal 团队开发。
它内置了 Tomcat、Jetty 等嵌入式服务器，支持自动配置和起步依赖。

## 主要特性
1. 自动配置 - 根据依赖自动配置 Spring 应用
2. 起步依赖 - 简化 Maven/Gradle 配置
3. 嵌入式服务器 - 内嵌 Tomcat/Jetty/Undertow
4. 健康检查 - Actuator 提供生产级监控
5. 外部化配置 - 支持 application.yml/properties

## 常用注解
- @SpringBootApplication - 启动类注解
- @RestController - RESTful 控制器
- @RequestMapping - 请求映射
- @Autowired - 依赖注入
- @Service - 服务层注解

# Redis 知识库

Redis 是一个开源的内存数据结构存储系统。
支持字符串、哈希、列表、集合、有序集合等数据类型。
常用于缓存、会话管理、消息队列、实时排行榜等场景。

## Redis 核心特性
- 内存存储，读写性能极高
- 支持数据持久化 (RDB/AOF)
- 支持主从复制和哨兵模式
- 支持集群模式
- 支持 Lua 脚本

## 常见命令
- SET/GET - 字符串操作
- HSET/HGET - 哈希操作
- LPUSH/LPOP/RPUSH/RPOP - 列表操作
- SADD/SMEMBERS - 集合操作
- ZADD/ZRANGE - 有序集合操作
